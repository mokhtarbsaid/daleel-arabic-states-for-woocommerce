A plugin help you to find the missed Arabic region countries states in Woocommerce.

== Description ==

This plugin helps you seamlessly integrate the missing Arabic region states into your WooCommerce store. With this enhancement, the missing states will appear correctly in essential areas such as the checkout page, shipping zones, and store address settings. This ensures a smoother shopping experience for Arabic customers regions and improves the accuracy of your store's localization settings."

Key Features:
* Add the missed Arabic countries states in Woocommerce..
* The missed arabic states appears in checkout page, shipping zones, store address.
* Follows WordPress.org plugin guidelines.



== Installation ==

1. Upload the `daleel-arabic-states-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The missed arabic states appears in checkout page, shipping zones.
