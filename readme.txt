== Description ==

A plugin help you to find the missed Arabic region countries cities in Woocommerce.
24 countries states with full translations.

Key Features:
* Add the missed Arabic countries states in Woocommerce..
* The missed arabic cities appears in checkout page, shipping zones, store address.
* you foinf the arabic translations in "languages" folder.
* Follows WordPress.org plugin guidelines.

This plugin is intended for developers and those looking to kickstart their WordPress plugin projects efficiently.

== Installation ==

1. Upload the `daleel-arabic-states-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The missed arabic states appears in checkout page, shipping zones.

= Can I use this in a production environment? =
Yes, but you may need to customize the code to fit your specific requirements.

= Is this plugin compatible with multisite? =
Yes, the plugin is compatible with WordPress multisite installations.
